## 🌟 Unique Design Elements

### 1. Glassmorphism Effect
Semi-transparent containers with blur effect creating premium appearance
```css
backdrop-filter: blur(10px);
border: 1px solid rgba(255, 255, 255, 0.3);
```

### 2. Gradient Text
Modern gradient text on all major headings
```css
-webkit-background-clip: text;
-webkit-text-fill-color: transparent;
background-clip: text;
```

### 3. Button Ripple Animation
Expanding circle effect on button hover
```css
::before {
    width: 0 → 300px on hover
    transition: 0.6s smooth
}
```

### 4. Smooth Transitions
Natural motion with cubic-bezier curves
```css
transition: all 0.35s cubic-bezier(0.4, 0, 0.2, 1);
```

### 5. Elevation Effects
Multiple shadow layers for depth perception
```css
box-shadow: 0 20px 60px rgba(0, 0, 0, 0.3),
            0 0 1px rgba(255, 255, 255, 0.5) inset;
```

## 🎯 How to Use

### View the Portal
```bash
# Portal is running at:
http://localhost:5000

# Login page:
http://localhost:5000/login

# Home page (after login):
http://localhost:5000/
```

### Customize Colors
Edit `static/style.css`:
```css
:root {
    --primary: #your-color;
    --secondary: #your-color;
    /* More variables */
}
```

### Modify Animations
Edit transition speeds in `static/style.css`:
```css
transition: all 0.35s cubic-bezier(...);
```

### Update Typography
Modify font sizes and weights in `static/style.css`:
```css
h1 { font-size: 38px; font-weight: 700; }
```
